Terminology Understanding

  1. Kafka
    • This is the aggregator which can get all the queries and different resources and process is accordingly
  2. Flume
    • This is the tool which can copy the log files in to Hadoop cluster. This is used for the streaming data
    • How it is different from Kafka?
      • in Kafka we are not talking about the type of the stream, it can always be the streaming data
  3. MapReduce
    • It's a simple form of parallel Processing which can provide the data quickly
  4. Hadoop
    • This is Big Data base
  5. HBase
    • Google's Big Table format, HDFS (Hadoop Distributed File System)
  6. Pig
    • This is another kind of database
  7. Hive
    • This is also another kind of database
  8. HDFS
    • Hadoop Data File system, This is basically the distributed fashion in which data gets stored. There will be replication factor of 3.
